                        throws java.lang.Exception
Split an animated GIF into individual frames and save them as GIF images. The split is "as is" as no frame decoding and other operations involved. As individual frames are often related to each other with regard to the animated GIF image, this sometimes leads to funny looking GIFs.

Parameters:
is - input GIF image stream
outputFilePrefix - optional output file name prefix
