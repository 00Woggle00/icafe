public class TGAReader
extends ImageReader

Decodes and shows 8 bit color mapped,black and white and 16,24 and 32 bit true color uncompressed and RLE TGA images. The compressed color mapped formats 32 and 33 are not implemented in this version because I have not found any information on these two formats.
